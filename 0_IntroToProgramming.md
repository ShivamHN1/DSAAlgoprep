# Intro to Programming with Java

Welcome! If this is your first programming lesson, you are exactly where you should be.

This page is just a quick, simple start so you understand the basic idea before we begin coding.

## What Is Programming?

Programming means telling a computer what to do, step by step.

In short:
- You have a problem.
- You break it into clear steps.
- You write those steps so the computer can follow them.

Those instructions are called a **program**.

## What Is a Programming Language?

A programming language is what we use to write those instructions.

Humans and computers do not "speak" the same way, so a programming language works like a bridge between us and the machine.

Examples of programming languages:
- Java
- Python
- C++
- JavaScript

Every language has its own syntax (writing rules), but the core thinking is similar across all languages.

## Why Are We Learning Java?

In this course, we will learn **Java**.

Java is a strong beginner language because it builds solid fundamentals:
- problem-solving
- clean code structure
- object-oriented thinking

It is also widely used in industry, interviews, and academics.

## Basic Java Terms You Should Know

Before we start coding, these Java terms will help:

### JDK (Java Development Kit)
The complete toolkit for developers.

It includes tools to write/compile code, plus the JRE.
If you want to build Java programs, you need the **JDK**.

### JRE (Java Runtime Environment)
The environment needed to run Java programs.

If you only want to run Java apps (not build them), JRE is enough.

### JVM (Java Virtual Machine)
The engine that actually runs Java bytecode.

Flow in simple words:
- You write Java code.
- Java code is compiled into bytecode.
- JVM runs that bytecode.

That is why Java is known for "write once, run anywhere" (with a compatible JVM).

## Quick Summary

- Programming is giving instructions to a computer.
- A programming language helps us write those instructions.
- In this course, we will learn Java.
- JDK is for building Java programs.
- JRE is for running Java programs.
- JVM executes Java bytecode.
